\subsubsection{Defining the Two Perspectives}
% GUIDE:
% Commercial /(bedriftsøkonomisk - HVA ER ET BRA ORD FOR DET?!?!?!): ash flows accruing to the operator only.
%   Revenue: spot arbitrage (buy cheap, store, displace peak);
%            reserve market payments (mFRR, aFRR capacity +
%            activation); peak load charge avoidance.
%   Cost: electricity purchase, grid tariff (effektledd), capital
%         servicing.
%   Excludes: value to the grid, value to other market participants,
%             environmental effects.
%
% Socio-economis /(Tror det er riktig for samfunnsøkonomisk??): total welfare including externalities.
%   Additional value drivers beyond what the operator captures:
%   - Avoided grid reinforcement (accrues to DSO/society via lower
%     tariffs)
%   - Freed grid capacity → faster connection of new loads
%   - System price effects (merit-order suppression during peak)
%   - Displaced fossil or high-cost peaking generation (CO₂ savings)
%   - Optionality value (resilience)
%
% Be explicit: these are not two measurements of the same thing;
% they are two accounting perimeters applied to the same asset.

\subsubsection{The Value Gap Mechanism}
% GUIDE: One tight paragraph. The mechanism:
%   (1) Thermal flexibility reduces system costs (avoided grid
%       investment, freed capacity, suppressed peak prices).
%   (2) Those cost reductions accrue to DSOs, TSO, and consumers —
%       not to the DH operator.
%   (3) The operator bears the full cost and faces tariff structures
%       that actively penalise flexible load.
%   (4) No transfer mechanism (flexibility payments, tariff
%       exemptions, aggregation credits) bridges the gap.
%   → Result: a structural underinvestment equilibrium. The gap is
%     not a market failure in the textbook sense; it is a regulatory
%     design choice that has not yet been updated to reflect the
%     changed system conditions.
% Cross-reference §2.3 (Barriers) for the specific mechanisms that
